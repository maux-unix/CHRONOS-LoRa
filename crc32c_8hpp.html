Detailed Description

CRC32-Castagnoli header using Hardware-accelerated & Parallel Computation CRC32C intrinsics from ARMv8 and ARMv9.

Provides fast CRC32C checksum computation through ARM ACLE intrinsics, specifically optimized for processors supporting CRC32 instructions.
