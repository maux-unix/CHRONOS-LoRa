Returns a view that formats range with elements separated by sep.

Example: 

auto v = std::vector<int>{1, 2, 3};
fmt::print("{}", fmt::join(v, ", "));
// Output: 1, 2, 3

fmt::join applies passed format specifiers to the range elements: 

fmt::print("{:02}", fmt::join(v, ", "));
// Output: 01, 02, 03
