Compute the CRC32C checksum of a memory span.

Uses ARM hardware-accelerated CRC32C byte intrinsics for efficient byte-wise checksum computation.

Initial CRC seed is 0xFFFFFFFF, and the final result is XOR'd with 0xFFFFFFFF per CRC32C standard conventions.

Debug information is printed before and after processing.

Parameters
frameThe memory frame to process.
Returns
The finalized CRC32C checksum.
Note
Requires ARMv8 or newer architecture with CRC32 instruction support.
